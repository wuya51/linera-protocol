§linera-alloy-signer-wallet

Local wallet implementations:

  • K256 private key
  • YubiHSM2

§Features

  • keystore: enables Ethereum keystore functionality on the LocalWallet type.
  • mnemonic: enables BIP-39 mnemonic functionality for building LocalWallets.
  • yubihsm: enables Wallets with YubiHSM2 support.
